# ⚽️ Football Oracle | 足球神谕

Welcome to the **magical and mysterious world** of the Football Oracle! This is a Deep Learning-powered football match predictor built with **Streamlit** and **TensorFlow**.

欢迎来到**神奇而神秘**的足球神谕世界！这是一个基于 **Streamlit** 和 **TensorFlow** 构建的深度学习足球比赛预测器。

---

## 🌟 Key Features | 核心功能

- **Club Search**: Intelligent search for home and away teams using names or codes.
  **球队搜索**：支持通过名称或代码智能搜索主客场球队。
- **Dynamic Feature Extraction**: Real-time calculation of form (Last 2/5 games), market values, and standings from **70,000+ historical records**.
  **动态特征提取**：从 **7万多条历史记录** 中实时计算近期战绩（近2/5场）、球员身价及联赛排名。
- **Deep Learning Inference**: Uses a 400KB optimized neural network pipeline for instant predictions.
  **深度学习推理**：使用 400KB 优化后的神经网络流水线，实现即时预测。
- **Interactive UI**: Progress animations and "miracle" celebrations for an engaging experience.
  **交互式界面**：包含进度条动画和“见证奇迹”的气球特效，提升用户体验。
